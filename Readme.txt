This project build a web application that scrapes information from various website related to the Mission to Mars

The directory Mission to Mars contains the following codes and files.

1. missiontomars.ipynb - jupyter notebook containing the scrapping code and this notebook has been converted into scrape_mars.py
2. app.py              - a flask app for rendering the index.html . This app needs to be run from the terminal and it will load the mission to mars html page.
3. scrape_mars.py      - an app containing the web scrapping and will trigger once "scrape mars data" button from html page has been pressed.
4. index.html          - code for html which diplay all the scrapped data.
5. Images folder        - saved screenshot of html before the scrapping and after the scrapping.
